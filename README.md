# DEC-Former

This is the original pytorch implementation of DEC-Former.

## Requirements
- python 3.7
- numpy == 1.19.2
- scipy == 1.7.3
- pandas == 1.1.5
- torch == 1.10.1
- scikit-learn==0.24.0


## Data
The dataset used in the paper can be downloaded from [BaiDuYun](https://pan.baidu.com/s/1jy-IUNTZUUYHNXtx7BpM-w?pwd=y1ef).
